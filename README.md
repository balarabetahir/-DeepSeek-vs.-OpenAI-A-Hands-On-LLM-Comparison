# -DeepSeek-vs.-OpenAI-A-Hands-On-LLM-Comparison
This project is a hands-on guide to exploring DeepSeek, a powerful open-source large language model (LLM). The guide covers using DeepSeek's web application, self-hosting it locally using Ollama, and comparing its performance against OpenAI's ChatGPT, particularly focusing on reasoning capabilities and token efficiency.
